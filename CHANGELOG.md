# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

### Added
- Initial public release (v0.1.0)
- Memory Palace generation from skin templates
- CLI: `new`, `skins`, `rooms`, `info` commands
- 5 bundled skins: clue, xfiles, digitalcircus, muppets, siliconvalley
- Integration with Cookiecutter for template rendering

### Fixed
- Configuration and API consistency for PyPI publication

## [0.1.0] - 2026-05-07

### Added
- Initial alpha release
- Filesystem-native Memory Palace framework
- Cookiecutter integration for palace generation
- CLI tool for palace management
- Type-safe Python implementation with mypy strict mode
- Comprehensive test suite
